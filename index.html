If you need to learn what some of the commands you can type actually do.

  • ls

    • list files in current directory

  • cat

    • print the contents of a file. stands for conCATenate

  • clear

    • clears out command history on screen

  • file

    • list metadata of a file

  • ssh

    • connect to an ip and open a remote shell

  • steghide

    • extract data hidden within a file
